# Sales Data Analysis Project
Set of real world data science tasks completed using the Python Pandas library.

🔍 Project Overview

This project analyzes real-world sales data to uncover key business insights such as top-performing products, best sales months, and customer purchasing behavior.

The analysis is performed using Python with libraries like Pandas and Matplotlib.

🛠️ Tools & Technologies
Python
Pandas
Matplotlib
Jupyter Notebook

📁 Dataset
Sample dataset (3 months) is included in this repository
Full dataset is not uploaded due to size limitations

⚙️ Key Steps Performed
🧹 Data Cleaning
Removed missing (NaN) values
Converted columns to correct data types
Filtered invalid rows
🏗️ Feature Engineering
Extracted Month from Order Date
Extracted City from Purchase Address
Created new columns for analysis

📊 Data Analysis
Monthly sales trend analysis
Sales by city
Most sold products
Products frequently sold together
Peak hours for customer purchases

📈 Key Insights
📌 San Francisco generated the highest revenue
📌 Sales peaked in December, likely due to holiday season
📌 AAA Batteries & AA Batteries were the most frequently sold products
📌 Evening hours showed higher purchasing activity
